## **KONU** INVITATIONS AND CELEBRATIONS - PART B

#### **BE GOING TO**

- We use **be going to + verb** (infinitive) to talk about future plans and predictions.
- We often use **future time expressions** with **be going to: tomorrow, next weekend, next**

#### Monday, etc.

I'm going to throw a party next Saturday.

#### **COUNTABLE / UNCOUNTABLE NOUNS**

- Countable (C) nouns are things you can count. They can be singular or plural.
  - e.g. one book, two books, three books.
- Uncountable (U) nouns are things you can't count. They can't be plural.
  - e.g. water, flour; NOT two waters, three flour

**Note**: Some nouns can be either C or U but they have different meanings.

e.g. Would you like some ice cream? (ice cream in general)

She wants an ice cream. (a cone of ice cream)

#### A / AN / SOME / ANY / A LOT OF

We use: - a I an with singular C nouns.

- some with plural C and U nouns in affirmative.
- any with plural C and U nouns in negatives and questions.
- a lot of with plural C and U nouns in affirmative.

**Attention:** When we ask for or offer things, we can use **some** in questions.

e.g. Would you like some tea?

Can I have **some** water?

We usually use *a lot of* in affirmative.

#### **HOW MUCH? / HOW MANY?**

We use *How much...?* with uncountable (U) nouns but *How many...?* with plural countable (C) nouns.

|                        |                                         | Answers                                                                                       |
|------------------------|-----------------------------------------|-----------------------------------------------------------------------------------------------|
| Uncountable (singular) | How much milk do you need?              | I need <b>some</b> milk.<br>I need <b>a lot</b> of milk.<br>I don't need <b>any</b> milk.     |
| Countable (plural)     | How many movies did you see last month? | I saw <b>some</b> movies.<br>I saw <b>a lot of</b> movies.<br>I didn't see <b>any</b> movies. |
