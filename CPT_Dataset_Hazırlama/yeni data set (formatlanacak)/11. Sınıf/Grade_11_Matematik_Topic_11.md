- II. bölgede sinüs fonksiyonu pozitif; kosinüs, tanjant ve kotanjant fonksiyonları negatif değerler alır.
- III. bölgede sinüs ve kosinüs fonksiyonları negatif, tanjant ve kontanjant fonksiyonları pozitif değerler alır.
- IV. bölgede kosinüs fonksiyonu pozitif; sinüs, tanjant ve kotanjant fonksiyonları negatif değerler alır.

Birinci Bölgede olan Açıların Trigonometrik Değerleri α nın birimi radvan olmak üzere

$$\cos\left(\frac{\pi}{2} - \alpha\right) = \sin\alpha \qquad \sin\left(\frac{\pi}{2} - \alpha\right) = \cos\alpha$$
$$\tan\left(\frac{\pi}{2} - \alpha\right) = \cot\alpha \qquad \cot\left(\frac{\pi}{2} - \alpha\right) = \tan\alpha$$

İkinci Bölgede olan Açıların Trigonometrik Değerleri

α nın birimi radvan olmak üzere

$$cos(\pi-\alpha) = -cos\alpha$$
  $sin(\pi-\alpha) = sin\alpha$   
 $tan(\pi-\alpha) = -tan\alpha$   $cot(\pi-\alpha) = -cot\alpha$ 

olur.

α nın birimi radyan olmak üzere

$$\cos\left(\frac{\pi}{2} + \alpha\right) = -\sin\alpha \quad \sin\left(\frac{\pi}{2} + \alpha\right) = \cos\alpha$$

$$\tan\left(\frac{\pi}{2} + \alpha\right) = -\cot\alpha \quad \cot\left(\frac{\pi}{2} + \alpha\right) = -\tan\alpha$$

Üçüncü Bölgede olan Açıların Trigonometrik Değerleri

α nın birimi radyan olmak üzere

$$cos(\pi+\alpha) = -cos\alpha$$
  $sin(\pi+\alpha) = -sin\alpha$   
 $tan(\pi+\alpha) = tan\alpha$   $cot(\pi+\alpha) = cot\alpha$  olur.

α nın birimi radyan olmak üzere

$$\cos(\frac{3\pi}{2} - \alpha) = -\sin\alpha \qquad \sin(\frac{3\pi}{2} - \alpha) = -\cos\alpha$$

$$\tan(\frac{3\pi}{2} - \alpha) = \cot\alpha \qquad \cot(\frac{3\pi}{2} - \alpha) = \tan\alpha$$
olur.

Dördüncü Bölgede olan Açıların Trigonometrik Değerleri

α nın birimi radyan olmak üzere

$$\cos(2\pi - \alpha) = \cos\alpha$$
  $\sin(2\pi - \alpha) = -\sin\alpha$   
 $\tan(2\pi - \alpha) = -\tan\alpha$   $\cot(2\pi - \alpha) = -\cot\alpha$ 

α nın birimi radyan olmak üzere

$$\cos\left(\frac{3\pi}{2} + \alpha\right) = \sin\alpha \quad \sin\left(\frac{3\pi}{2} + \alpha\right) = -\cos\alpha$$

$$\tan\left(\frac{3\pi}{2} + \alpha\right) = -\cot\alpha \cot\left(\frac{3\pi}{2} + \alpha\right) = -\tan\alpha$$
