#### ÜCGEN ESITSIZLIĞI

![](_page_0_Picture_12.jpeg)

Bir ücgende iki kenar uzunluğunun toplamı, üçüncü kenarın uzunluğundan büyüktür. ABC üçgeninde;

#### $\rightarrow$ a < b+c

$$>$$
 b < a + c

$$\rightarrow$$
 c < a+b olur.

Bir üçgende herhangi bir kenar uzunluğu, diğer iki kenar uzunluğu toplamından kücük: farklarının mutlak değerinden büyüktür. Bu bağıntıya üçgen eşitsizliği denir. ABC ücgeninde;

$$|b-c| < a < b+c$$

$$\begin{vmatrix} b-c \\ -c \end{vmatrix} < a < b+c \\ -c \end{vmatrix} < b < a+c \end{vmatrix}$$

$$\left| a-b \right| < c < a+b$$
 olur.

![](_page_0_Picture_21.jpeg)
