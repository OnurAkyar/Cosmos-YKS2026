**ÇEMBER DENKLEMİ**

**DENKLEMLERİ VERİLEN DOĞRU İLE ÇEMBERİN BİR-BİRİNE GÖRE DURUMLARI** 

![](_page_0_Picture_6.jpeg)

|MH| > r ise doğru çemberi kesmez.

![](_page_0_Picture_8.jpeg)

|MH| = r ise doğru çembere teğettir.

![](_page_0_Picture_10.jpeg)

|MH| < r ise doğru çemberi iki noktada keser.

Genel denklemi x<sup>2</sup> +y<sup>2</sup> + Dx + Ey + F = 0 olan çember ile y = mx + n doğrusunun ortak çözümü yapılarak doğru ile çemberin birbirine göre durumları incelenir.

Çember denkleminde y yerine mx + n yazılarak

ax<sup>2</sup> + bx + c = 0 biçiminde ikinci dereceden bir denklem elde edilir. Bu denklemin diskriminantı Δ = b<sup>2</sup> – 4ac olmak üzere

- 1. Δ = b<sup>2</sup> – 4ac < 0 ise elde edilen denklemin kökü yoktur. Bu durumda **doğru çemberi kesmez**.
- 2. Δ = b<sup>2</sup> – 4ac = 0 ise elde edilen denklemin çift katlı kökü vardır. Bu durumda doğru **çembere teğettir**.
- 3. Δ = b<sup>2</sup> – 4ac > 0 ise elde edilen denklemin farklı iki kökü vardır. Bu durumda **doğru çemberi iki noktada keser**.
