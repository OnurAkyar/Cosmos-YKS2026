### LOGARİTMİK DENKLEMLER

 $a \in \mathbb{R}^+ - \{1\}$  ve  $b \in \mathbb{R}$  olmak üzere

$$> \log_a f(x) = b \Leftrightarrow f(x) = a^b$$

$$\triangleright \log_a f(x) = \log_a g(x) \Leftrightarrow f(x) = g(x)$$

$$> \log_{f(x)} g(x) = b \Leftrightarrow g(x) = (f(x))^b$$

$$(f(x) > 0, g(x) > 0 \text{ ve } f(x) \neq 1)$$

Üstel esitsizliklerde  $a^{f(x)} \geq a^{g(x)}$  için

$$\Rightarrow$$
 a >1 ise  $f(x) \ge g(x)$  olur.

$$> 0 < a < 1$$
 ise  $f(x) \le g(x)$  olur.

Logaritmik eşitsizliklerde  $\log_a f(x) \le \log_a g(x)$  için

$$\Rightarrow$$
 a >1 ise  $f(x) \le g(x)$  olur.

$$> 0 < a < 1$$
 ise  $f(x) \ge g(x)$  olur.
