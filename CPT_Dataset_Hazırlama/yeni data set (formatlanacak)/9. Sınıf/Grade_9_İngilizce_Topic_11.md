## KONU

*I can dance well.*

*He can lift heavy weights.*

*We use can't to talk about inabilities.*

*She can't read or write. They can't ride a bike.*

| Affirmative        |     |        |  |
|--------------------|-----|--------|--|
| Subject            | can | Verb   |  |
| I /You / We / They | can | dance. |  |
| / He / She / It    |     |        |  |

| Subject            | can not (can't) | Verb   |
|--------------------|-----------------|--------|
| I /You / We / They | can't           | dance. |
| / He / She / It    |                 |        |

| Can | Subject                                   | Verb | Short answer                                                                                                                                                                        |
|-----|-------------------------------------------|------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Can | I / you / we<br>/ they / he<br>/ she / it |      | Yes, you can. / No, you can't. Yes, I can. / No, I can't. Yes, we can. / No, we can't. Yes, they can. / No, they can't. Yes, he can. / No, he can't. Yes, she can. / No, she can't. |
|     |                                           |      | Yes, it can. / No, it can't.                                                                                                                                                        |
