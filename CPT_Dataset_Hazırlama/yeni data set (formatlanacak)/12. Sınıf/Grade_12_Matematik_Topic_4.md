# **KONU** LOGARITMA FONKSIYONUNUN GRAFIĞI

# **Logaritma Fonksiyonunun Grafiği:**

f: R → R<sup>+</sup> , a > 1 için f(x) = ax fonksiyonunun tersi f –1 (x) = loga x fonksiyonudur. Birbirinin tersi olan iki fonksiyonun grafiği y = x doğrusuna göre simetrik olduğundan önce y = x doğrusu çizilir.

Daha sonra f(x) = ax fonksiyonunun grafiğinin y = x doğrusuna göre simetriği alınarak f –1 (x) = loga x fonksiyonunun grafiği oluşturulur.

![](_page_0_Figure_6.jpeg)

![](_page_0_Figure_7.jpeg)

a ∈ R+ – {1} Olmak üzere f: R<sup>+</sup>→ R, f(x) = loga x logaritma fonksiyonu, a > 1 için artan olup grafiği yandaki gibidir.

![](_page_0_Figure_9.jpeg)

a ∈ R+ – {1} Olmak üzere f: R<sup>+</sup>→ R, f(x) = loga x logaritma fonksiyonu, 0 < a < 1 için azalan olup grafiği yandaki gibidir.
